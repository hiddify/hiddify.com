      • Щ…Ш¬ШІШ§ Ъ©ШұШҜЩҶ ШӘШұШ§ЩҒЫҢЪ© ШҜШ§Ш®Щ„ЫҢ ЫҢШ§ Щ…ШіШӘЩӮЫҢЩ… Ъ©ШұШҜЩҶ ШӘШұШ§ЩҒЫҢЪ© ШіШ§ЫҢШӘвҖҢЩҮШ§ЫҢ ШҜШ§Ш®Щ„ЫҢ
        • ШЁШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШӘЩҶШёЫҢЩ…Ш§ШӘ System proxy
        • Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШӘЩҶШёЫҢЩ…Ш§ШӘ Routing

    ШўЩ…ЩҲШІШҙ Ъ©Ш§Шұ ШЁШ§ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ HiddifyN

    Ш§ЫҢЩҶ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ ШЁШұ ЩҫШ§ЫҢЩҮ Xray ШӘЩҲШіШ№ЩҮ ШҜШ§ШҜЩҮ ШҙШҜЩҮ Ш§ШіШӘ ЩҲ ШҜШ§ШұШ§ЫҢ ЩӮШ§ШЁЩ„ЫҢШӘвҖҢЩҮШ§ЫҢ ЩҒШұШ§ЩҲШ§ЩҶЫҢ Ш§ШіШӘ Ъ©ЩҮ ШҜШұ Ш§ШҜШ§Щ…ЩҮ ШЁШұШұШіЫҢ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ.

    ШҜШ§ЩҶЩ„ЩҲШҜ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ

    ШЁШұШ§ЫҢ ШҜШ§ЩҶЩ„ЩҲШҜ ШЁШұЩҶШ§Щ…ЩҮ ШҜШұ ЩҫЩҶЩ„ Ъ©Ш§ШұШЁШұЫҢ Ш®ЩҲШҜШҢ ШӘШЁ Щ…ШұШЁЩҲШ· ШЁЩҮ ШҜШіЪ©ШӘШ§Щҫ ШұШ§ ШЁШ§ШІ Ъ©ЩҶЫҢШҜ ЩҲ ШЁШ®Шҙ Щ…ШұШЁЩҲШ· ШЁЩҮ HiddifyN ШұШ§ ШЁШ§ШІ Ъ©ЩҶЫҢШҜ ЩҲ ШіЩҫШі ШұЩҲЫҢ ШўЫҢЪ©ЩҲЩҶ ЩҲЫҢЩҶШҜЩҲШІ (ШҜЪ©Щ…ЩҮ ШҙЩ…Ш§ШұЩҮ Ыі) ШЁШІЩҶЫҢШҜ ШӘШ§ ШЁШұЩҶШ§Щ…ЩҮ ШҜШ§ЩҶЩ„ЩҲШҜ ШҙЩҲШҜ.

    ЩҶШөШЁ ШЁШұЩҶШ§Щ…ЩҮ

    ЩҶШөШЁ ШЁШұЩҶШ§Щ…ЩҮ Ш®ЫҢЩ„ЫҢ ШіШ§ШҜЩҮ Ш§ШіШӘ. Ъ©Ш§ЩҒЫҢШіШӘ Ш§ШЁШӘШҜШ§ ШЁШұЩҶШ§Щ…ЩҮ Runtime 6 ШұШ§ ЩҶШөШЁ Ъ©ЩҶЫҢШҜ ЩҲ ШіЩҫШі ЩҒШ§ЫҢЩ„ ШҜШ§ЩҶЩ„ЩҲШҜ ШҙШҜЩҮ ШұШ§ Ш§Ъ©ШіШӘШұЪ©ШӘ Ъ©ШұШҜЩҮ ЩҲ ШҜШұ ЩҫЩҲШҙЩҮ Щ…ШұШЁЩҲШ· ШЁЩҮ ШўЩҶ ЩҒШ§ЫҢЩ„ HiddifyN ШұШ§ Ш§Ш¬ШұШ§ ЩҶЩ…Ш§ЫҢЫҢШҜ.

    Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ ЩҫШұЩҲЩҒШ§ЫҢЩ„ ШЁЩҮ ШЁШұЩҶШ§Щ…ЩҮ

    ШҜШұ ЩҫЩҶЩ„ Ъ©Ш§ШұШЁШұЫҢ Щ…Ш¬ШҜШҜШ§ ШЁЩҮ ШЁШ®Шҙ HiddifyN ШЁШұЩҲЫҢШҜ ЩҲ ШұЩҲЫҢ Ъ©Щ„ЫҢЪ© ШЁШұШ§ЫҢ Ш§ЩҒШІЩҲШҜЩҶ Ъ©Ш§ЩҶЩҒЫҢЪҜ ШЁШІЩҶЫҢШҜ ШӘШ§ Ъ©Ш§ЩҶЩҒЫҢЪҜвҖҢЩҮШ§ ШЁЩҮ ШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ ШЁЩҮ ШЁШұЩҶШ§Щ…ЩҮ Ш§Ш¶Ш§ЩҒЩҮ ШҙЩҲЩҶШҜ.

    ШӯШ§Щ„ШӘвҖҢЩҮШ§ЫҢ ЩҫШұЩҲЪ©ШіЫҢ

    ЫҢЪ©ЫҢ Ш§ШІ Щ…ЩҮЩ…вҖҢШӘШұЫҢЩҶ ШЁШ®ШҙвҖҢЩҮШ§ЫҢ Ш§ЫҢЩҶ Ш§ЩҫЩ„ЫҢЪ©ЫҢШҙЩҶШҢ ШӯШ§Щ„ШӘвҖҢЩҮШ§ЫҢ ЩҫШұЩҲЪ©ШіЫҢ Ш§ШіШӘ Ъ©ЩҮ ШҜШұ Ш§ШҜШ§Щ…ЩҮ ШҜШұШЁШ§ШұЩҮ ЩҮШұ ЩӮШіЩ…ШӘ ШўЩҶ ШӘЩҲШ¶ЫҢШӯ ШҜШ§ШҜЩҮ Ш®ЩҲШ§ЩҮШҜ ШҙШҜ.

    Auto/Smart

    ШЁЩҮ ШөЩҲШұШӘ Ш®ЩҲШҜЪ©Ш§Шұ ШЁЩҮ Ъ©Ш§ЩҶЩҒЫҢЪҜЫҢ ШЁШ§ ШЁЫҢШҙШӘШұЫҢЩҶ ШіШұШ№ШӘ (Ъ©Щ…ШӘШұЫҢЩҶ ЩҫЫҢЩҶЪҜ) Щ…ШӘШөЩ„ Щ…ЫҢвҖҢШҙЩҲШҜ.

    Load-Balance

    ШЁЩҮ ШөЩҲШұШӘ ЩҮЩ…ШІЩ…Ш§ЩҶ ШЁЩҮ ЪҶЩҶШҜ Ъ©Ш§ЩҶЩҒЫҢЪҜ ЩҲШөЩ„ Щ…ЫҢвҖҢШҙЩҲШҜ ЩҲ ШӘШұШ§ЩҒЫҢЪ© ШЁЫҢЩҶ ШўЩҶвҖҢЩҮШ§ ЩҫШ®Шҙ Щ…ЫҢвҖҢШҙЩҲШҜ. Ш§ЫҢЩҶ ШӯШ§Щ„ШӘ ШЁШұШ§ЫҢ ШІЩ…Ш§ЩҶЫҢ Ш®ЫҢЩ„ЫҢ Щ…ЩҒЫҢШҜ Ш§ШіШӘ Ъ©ЩҮ ШҙЩ…Ш§ ЪҶЩҶШҜ Ъ©Ш§ЩҶЩҒЫҢЪҜ ШЁШ§ ШўЫҢЩҫЫҢ Ъ©Ш«ЫҢЩҒ ШҜШ§ШұЫҢШҜШҢ ШЁШ§ ЩҫШ®Шҙ ШЁШ§Шұ ШұЩҲЫҢ ЪҶЩҶШҜ Ъ©Ш§ЩҶЩҒЫҢЪҜ ШЁЩҮ ШөЩҲШұШӘ ЩҮЩ…ШІЩ…Ш§ЩҶШҢ Ш№Щ…Щ„Ш§ ШіШұШ№ШӘ ЩӮШ§ШЁЩ„ ЩӮШЁЩҲЩ„ЫҢ ШЁЩҮ ШҜШіШӘ Щ…ЫҢвҖҢШўЫҢШҜ.

    Manual

    ШЁШұШ§ЫҢ ШӘЩҶШёЫҢЩ… Ъ©Ш§ЩҶЩҒЫҢЪҜвҖҢЩҮШ§ ШЁЩҮ ШөЩҲШұШӘ ШҜШіШӘЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ. ШЁШ§ Ъ©Щ„ЫҢЪ© ШұЩҲЫҢ ШўЩҶШҢ ШӘЩҶШёЫҢЩ…Ш§ШӘ ЩҫЫҢШҙШұЩҒШӘЩҮ ШЁШұЩҶШ§Щ…ЩҮ ШЁШ§ШІ Щ…ЫҢвҖҢЪҜШұШҜШҜ Ъ©ЩҮ ШҜШұ Ш§ШҜШ§Щ…ЩҮ ШЁЫҢШҙШӘШұ ШӘЩҲШ¶ЫҢШӯ ШҜШ§ШҜЩҮ Ш®ЩҲШ§ЩҮШҜ ШҙШҜ.

    ШӘШәЫҢЫҢШұ ШІШЁШ§ЩҶ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ

    ШЁШұШ§ЫҢ ШӘШәЫҢЫҢШұ ШІШЁШ§ЩҶ ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ ШұЩҲЫҢ Щ…ЩҶЩҲЫҢ Щ…ШҙШ®Шө ШҙШҜЩҮ ШҜШұ ШӘШөЩҲЫҢШұ ШІЫҢШұ ШЁШІЩҶЫҢШҜ ЩҲ ШІШЁШ§ЩҶ ШұШ§ Ш§ЩҶШӘШ®Ш§ШЁ Ъ©ЩҶЫҢШҜ. ШіЩҫШі ЫҢЪ© ШЁШ§Шұ ШЁШұЩҶШ§Щ…ЩҮ ШұШ§ ШЁШЁЩҶШҜЫҢШҜ ЩҲ Щ…Ш¬ШҜШҜШ§ ШЁШ§ШІ Ъ©ЩҶЫҢШҜ.

    ШӘШәЫҢЫҢШұ ШӘЩ… ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ

    ШЁШұШ§ЫҢ ШӘШәЫҢЫҢШұ ШӘЩ… ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ ШұЩҲЫҢ ШҜЪ©Щ…ЩҮ Щ…ШҙШ®Шө ШҙШҜЩҮ Ъ©Щ„ЫҢЪ© Ъ©ЩҶЫҢШҜ.

    Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ЩҫШұЩҲЩҒШ§ЫҢЩ„ Ъ©Ш§ШұШЁШұЫҢ

    ШҜШұ ШЁШ®Шҙ ЩҶЩ…Ш§ЫҢШҙ ШҜШ§ШҜЩҮ ШҙШҜЩҮ ШҜШұ ШІЫҢШұ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Щ…ШұШЁЩҲШ· ШЁЩҮ ЩҫШұЩҲЩҒШ§ЫҢЩ„ Ъ©Ш§ШұШЁШұЫҢ Ш§ЫҢЩ…ЩҫЩҲШұШӘ ШҙШҜЩҮ Ш§ШІ Ш¬Щ…Щ„ЩҮ ШІЩ…Ш§ЩҶ Ш§ЩҶЩӮШ¶Ш§ ЩҲ Щ…ЫҢШІШ§ЩҶ ШӯШ¬Щ… Ъ©Щ„ЫҢ ЩҲ ЩҮЩ…ЫҢЩҶШ·ЩҲШұ ШӯШ¬Щ… Щ…ШөШұЩҒ ШҙШҜЩҮ ЩҶЩ…Ш§ЫҢШҙ ШҜШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.

    ШҜШұ Ш§ЫҢЩҶ ЩӮШіЩ…ШӘ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ЩҫШұЩҲЩҒШ§ЫҢЩ„вҖҢЩҮШ§ЫҢ Ш¬ШҜЫҢШҜ ШұШ§ ШЁЩҮ ШөЩҲШұШӘ ШҜШіШӘЫҢ ЩҶЫҢШІ Ш§Ш¶Ш§ЩҒЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ. Ъ©Ш§ЩҒЫҢШіШӘ ШұЩҲЫҢ Import ШЁШІЩҶЫҢШҜ ЩҲ Щ„ЫҢЩҶЪ© Ъ©ЩҫЫҢ ШҙШҜЩҮ Ш§ШІ ЩҫЩҶЩ„ ШұШ§ Ъ©Ш§ШұШЁШұЫҢ ШұШ§ Ш§Ш¶Ш§ЩҒЩҮ ЩҶЩ…Ш§ЫҢЫҢШҜ.

    Ш§ШӘШөШ§Щ„ ШЁЩҮ ЩҒЫҢЩ„ШӘШұШҙЪ©ЩҶ

    ШЁШұШ§ЫҢ Ш§ШӘШөШ§Щ„ ШЁЩҮ ЩҒЫҢЩ„ШӘШұШҙЪ©ЩҶ ШЁШ§ЫҢШҜ ШұЩҲЫҢ ШҜЪ©Щ…ЩҮ ШҜШ§ЫҢШұЩҮвҖҢШ§ЫҢ ЩҲШіШ· Ъ©Щ„ЫҢЪ© Ъ©ЩҶЫҢШҜ. ЩҶШұЩ…вҖҢШ§ЩҒШІШ§Шұ ШЁШұ Ш§ШіШ§Ші ШӯШ§Щ„ШӘ ЩҫШұЩҲЪ©ШіЫҢ Ш§ЩҶШӘШ®Ш§ШЁ ШҙШҜЩҮ ШЁЩҮ Ъ©Ш§ЩҶЩҒЫҢЪҜвҖҢЩҮШ§ Щ…ШӘШөЩ„ Щ…ЫҢвҖҢЪҜШұШҜШҜ ЩҮЩ…ЪҶЩҶЫҢЩҶ ЩҫЫҢЩҶЪҜ Щ…ШұШЁЩҲШ· ШЁЩҮ Ъ©Ш§ЩҶЩҒЫҢЪҜ ШҜШұ Ъ©ЩҶШ§Шұ Ш§ЫҢЩҶ ШҜЪ©Щ…ЩҮ ЩҶЩ…Ш§ЫҢШҙ ШҜШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜ.

    ШӘЩҶШёЫҢЩ…Ш§ШӘ ЩҫЫҢШҙШұЩҒШӘЩҮ ШЁШұЩҶШ§Щ…ЩҮ

    Ш§ЫҢЩҶ ШЁШ®Шҙ ШЁШ§ ШІШҜЩҶ ШҜЪ©Щ…ЩҮ Manual ШҜШұ ЩӮШіЩ…ШӘ ШӯШ§Щ„ШӘ ЩҫШұЩҲЪ©ШіЫҢ ШЁШ§ШІ Щ…ЫҢвҖҢШҙЩҲШҜ.

    Ыұ. Settings

     ШӘЩҶШёЫҢЩ…Ш§ШӘ ЩҫЫҢШҙШұЩҒШӘЩҮ ШЁШұЩҶШ§Щ…ЩҮ ШҜШұ Ш§ЫҢЩҶ ЩӮШіЩ…ШӘ ЩӮШұШ§Шұ ШҜШ§ШұШҜ.
    Option setting
     ШҜШұ Ш§ЫҢЩҶ ШІЫҢШұШЁШ®Шҙ ШӘЩҶШёЫҢЩ…Ш§ШӘ Ш§Ш®ШӘЫҢШ§ШұЫҢ ШЁШұЩҶШ§Щ…ЩҮ ЩӮШұШ§Шұ ШҜШ§ШұШҜ Ъ©ЩҮ Ш®ЩҲШҜ ШҙШ§Щ…Щ„ ЪҶЩҶШҜ ШЁШ®Шҙ Щ…ШӘЩҒШ§ЩҲШӘ Ш§ШіШӘ. * ШӘЩҶШёЫҢЩ…Ш§ШӘ ЩҫШ§ЫҢЩҮ Ъ©ЩҮ ШЁШұШ§ЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ ШұЩҲШІЩ…ШұЩҮ ЪҶЩҶШҜШ§ЩҶ ЩҶЫҢШ§ШІ ШЁЩҮ ШӘШәЫҢЫҢШұ ШҜШұ Ш§ЫҢЩҶ ЩӮШіЩ…ШӘ ЩҶЫҢШіШӘ. ШӘЩҶЩҮШ§ ШўЩҫШҙЩҶ ЩҫШұЪ©Ш§ШұШЁШұШҜ ШҙШ§ЫҢШҜ ЩҫЩҲШұШӘ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙШҜЩҮ ШЁШұШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ ШЁШ§ШҙШҜ Ъ©ЩҮ ШҜШұ Ш§ЫҢЩҶ ШЁШ®Шҙ ЩӮШ§ШЁЩ„ ШӘШәЫҢЫҢШұ Ш§ШіШӘ.
    Ш§ЪҜШұ ЩҶЫҢШ§ШІ ШЁЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҫШұЩҲЪ©ШіЫҢ ШҜШ§ШұЫҢШҜ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ЩҫШұЩҲЪ©ШіЫҢ socks5 ШұШ§ ШұЩҲЫҢ Ш§ЫҢЩҶ ЩҫЩҲШұШӘ ШӘЩҶШёЫҢЩ… ЩҶЩ…Ш§ЫҢЫҢШҜ. * ШӘЩҶШёЫҢЩ…Ш§ШӘ V2rayN ШҜШұ Ш§ЫҢЩҶ ШЁШ®Шҙ ШӘЩҶШёЫҢЩ…Ш§ШӘ ШҜЫҢЩҒШ§Щ„ШӘ ШЁШұЩҶШ§Щ…ЩҮ V2rayN Ъ©ЩҮ ЩҫШ§ЫҢЩҮ Ш§ШөЩ„ЫҢ ШЁШұЩҶШ§Щ…ЩҮ HIddifyN Ш§ШіШӘШҢ ЩӮШұШ§Шұ ШҜШ§ШұШҜ.
